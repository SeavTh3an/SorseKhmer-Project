Sorse Khmer is a Unity-based educational typing game designed to help players practice typing in the Khmer language.
The game combines learning with action-based gameplay, where players must type Khmer words correctly to defeat approaching enemies.

This project was developed as a final project under academic guidance and focuses on improving typing accuracy, language familiarity, and reaction speed.
